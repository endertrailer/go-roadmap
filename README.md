# Go Course Path

## 🟢 Phase 1: Go Foundations
- Variables, types, functions, loops, conditionals
- Structs & methods
- Interfaces
- Error handling
- Concurrency (goroutines, channels, sync.WaitGroup)

**Projects:**
- Number guessing game
- File word counter
- CSV parser
- **Phase Project:** Pomodoro timer CLI

---

## 🟢 Phase 2: Go for Backend APIs
- net/http standard library
- JSON handling (encoding/json)
- Routing (Gin or Fiber)
- REST APIs + PostgreSQL
- Authentication (JWT, middleware, sessions)
- Testing

**Projects:**
- URL shortener
- Pastebin clone
- **Phase Project:** Blog API

---

## 🟢 Phase 3: Databases & Persistence
- SQL queries, joins, transactions
- PostgreSQL (pgx)
- Connection pooling
- ORMs (GORM or sqlx)
- Migrations (golang-migrate)

**Projects:**
- CLI expense tracker
- Extend URL shortener with PostgreSQL
- **Phase Project:** Job board backend

---

## 🟢 Phase 4: Infrastructure Basics
- Networking basics (TCP/IP, HTTP, DNS)
- Docker & Docker Compose
- Logging & configuration (.env, structured logs)
- CI/CD (GitHub Actions)

**Projects:**
- Dockerize URL shortener
- GitHub Actions pipeline for Blog API
- **Phase Project:** CLI system monitor

---

## 🟢 Phase 5: Cloud & Deployment
- Kubernetes basics
- Cloud providers (AWS, GCP, or DigitalOcean)
- Monitoring & metrics (Prometheus, Grafana, OpenTelemetry)
- Helm

**Projects:**
- Deploy URL shortener
- Metrics endpoint for Blog API
- **Phase Project:** Job board deployment with monitoring

---

## 🟢 Phase 6: Advanced Topics
- Redis (caching, rate limiting, pub/sub)
- gRPC (protocol buffers)
- Message queues (Redis pub/sub, NATS)
- Distributed systems concepts
- Scaling patterns

**Projects:**
- Redis rate limiter
- Real-time notification service
- **Phase Project:** Distributed task queue
